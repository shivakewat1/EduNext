# 🎓 EduNext — Student & Teacher Dashboard System

EduNext is a role-based web application designed to streamline assignment management between teachers and students. It enables teachers to create and manage assignments while allowing students to view, complete, and submit them efficiently through a centralized dashboard.

---

## 🚀 Core Features

### 🔐 Authentication & Authorization

* Secure login and signup for Students and Teachers
* Role-based access control
* Protected routes for dashboards

### 👨‍🏫 Teacher Dashboard

* Create and manage assignments
* Set deadlines for submissions
* View and track student submissions
* Monitor assignment status (Submitted / Pending)

### 👨‍🎓 Student Dashboard

* Access assigned tasks
* Submit assignments online
* Track submission status
* Explore available courses

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** CSS / Tailwind CSS
* **Backend:** Firebase / Node.js & Express
* **Database:** Firestore / MongoDB

---

## 🔄 Application Workflow

1. Users register as a Student or Teacher
2. Authentication verifies user credentials
3. Role-based dashboard is rendered
4. Teachers create and assign tasks with deadlines
5. Students submit assignments within the platform
6. Teachers review and track submissions
